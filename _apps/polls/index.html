GitHub Polls Bot

A GitHub App built with Probot that automatically creates gh-polls in GitHub issues.

Usage

  1. Configure the GitHub App: github.com/apps/polls
  2. Add command to issue:
/polls Option1 'Option 2' "Option 3"
